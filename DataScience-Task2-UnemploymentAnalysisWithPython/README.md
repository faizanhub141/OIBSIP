# 📖 Project Overview

This project focuses on analyzing unemployment data using Python to identify trends, patterns, and regional differences in unemployment rates. The analysis was performed using data visualization and statistical techniques to gain meaningful insights from the dataset.

---

# 🎯 Objective

The main objective of this project is to:
- Analyze unemployment rates across different regions.
- Understand unemployment trends over time.
- Compare employment and unemployment metrics.
- Visualize regional and zonal unemployment patterns.
- Perform correlation analysis to identify relationships between variables.

---

# 📂 Dataset

**Source:** Kaggle

### Dataset Columns
- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Region.1

- Latitude
